# Perception Zero - Official Site

Static HTML/CSS website for the band Perception Zero, styled with a 2000s aesthetic.

## Features

- Classic 2000s web design with dark theme
- Responsive layout
- Pages: Home, About, Music, Shows, Gallery, Contact
- Marquee text effects
- Retro navigation and styling

## Deployment

Ready for Netlify deployment - just push to GitHub and connect to Netlify.

## Pages

- **index.html** - Homepage with news and announcements
- **about.html** - Band bio and member information
- **music.html** - Discography and streaming links
- **shows.html** - Tour dates and booking information
- **gallery.html** - Photo gallery with placeholder images
- **contact.html** - Contact information and mailing list

## Customization

Replace placeholder content with your actual:
- Band photos
- Music embeds (Spotify, Bandcamp, YouTube)
- Real tour dates
- Social media links
- Contact information
